# 朱老板
  因为总是记不住每个菜和商品的价格所以诞生了这个系统，协助记忆菜和商品的价格。其实是给自己开发的，每次回家要卖菜，但是价格总是记不住。所以空闲做了这个app，基于meteor开发的，可以使用cordova打包成App。
